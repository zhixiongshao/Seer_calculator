# 赛尔号 · 精灵能力值计算器

基于官方培养机制公式，计算精灵六项基础能力值。

## 使用方式

直接用浏览器打开 `index.html` 即可，无需安装依赖。

```bash
open index.html
```

## 精灵图鉴数据

精灵种族值来自 [SeerAPI](https://api.seerapi.com)（解析游戏客户端数据，目前约 **4896** 只精灵）。

- 打开页面时会**自动检查更新**，周五新精灵上线后会增量同步
- 离线时使用本地 `data/pets.js` 缓存

手动全量同步（可选）：

```bash
node scripts/fetch-pets.js
```

## 功能

- **搜索选择精灵**，自动填充种族值（名称或序号）
- 输入种族值、学习力、天赋、等级、性格
- **称号 / 刻印（3 槽）/ 套装**加成，数据来自 SeerAPI
- 战队加成、年费、体力上限
- 公式与 [4399 官方计算器](https://news.4399.com/seer/jsq/) 一致
- 实时计算六项能力值
- 显示各项贡献拆解
- 满级时提示推荐学习力（254/255 刷法）
- 学习力总和校验（上限 510）

更新加成数据：

```bash
node scripts/fetch-bonuses.js
```

## 公式

**攻击 / 防御 / 特攻 / 特防 / 速度：**

```
floor(((种族×2 + 个体 + 努力值÷4) × 等级÷100 + 5) × 性格修正)
```

**体力（不受性格影响）：**

```
floor((种族×2 + 个体 + 努力值÷4) × 等级÷100 + 等级 + 10)
```
